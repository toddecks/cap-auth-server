# CSP Pro Email Templates

These templates are loaded by `server.js` for form notification emails and preview pages.

Supported placeholders:

- `{{name}}` escapes the value before inserting it.
- `{{{name}}}` inserts trusted HTML generated by the backend helpers.

Files:

- `layout.html` wraps every email.
- `shift_report.html` controls the Shift Report body.
- `forklift_inspection.html` controls the Forklift Inspection body.
- `crane_inspection.html` controls the Crane Inspection body.
- `operational_inspection.html` controls the Operational Inspection body.

Preview locally:

```text
http://localhost:3000/api/pro/forms/email-preview
```
